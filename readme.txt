== Installation ==

The plugin is very simple to install:

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all! Enjoy!

== Frequently Asked Questions ==

= Does the plugin work on all browsers? =

The plugin has been tested on IE7+, Firefox 3.6, Opera 11.0 and Safari 5.0. It should work on previous versions of theses browsers.
